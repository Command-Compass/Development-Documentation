# Contributing to Project

We welcome contributions to our project! Please read through these guidelines before contributing.

## How to Contribute

1. **Create a branch**
2. **Make your changes**
3. **Commit your changes**
4. **Push to your branch**
5. **Create a Pull Request**

## Code of Conduct

Please read and follow our [Code of Conduct](CODE_OF_CONDUCT.md).

## Coding Standards

Please adhere to the coding standards outlined in [Coding Standards](guides/development/coding-standards.md).

## Pull Request Process

1. Ensure your changes pass all tests and linters.
2. Provide a clear description of your changes.
3. Link to any relevant issues.
4. Ensure your branch is up to date with the base branch.

## Reporting Issues

Please use the [GitHub Issues](https://github.com/your-org/repo/issues) to report any bugs or feature requests.

Thank you for contributing!
